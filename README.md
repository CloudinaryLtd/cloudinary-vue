# Cloudinary Vue SDK

## Usage

### Installation

Install the library with npm

```
npm i cloudinary-vue
```

or refer to it using unpkg CDN

```html
<script src="//unpkg.com/cloudinary-vue/dist/Cloudinary.umd.min.js">
```

## Development

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

### Run your unit tests

```
npm t
```
